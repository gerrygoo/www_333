# Technology Stack

## Core Technologies
*   **HTML5:** Semantic structure for the landing page and content sub-pages.
*   **CSS3:** Custom styles following a mobile-first approach, utilizing advanced filters and animations for "digital decay" effects.
*   **JavaScript (Vanilla):** Minimal use for performance-aware scaling of visual effects and subtle interactive glitches.

## Libraries & Reset
*   **Normalize.css:** Standard CSS reset to ensure cross-browser consistency.

## Assets & Typography
*   **Custom Local Fonts:** `Terminal Grotesque`, `Overconsumption`, `Mustasurma`, `WordsWithA`, `YellowWhiteOpozzum`.
*   **External Media:** Large video assets (e.g., demo reels) will be hosted externally to optimize load times and repository size.

## Hosting & Deployment
*   **GitHub Pages:** Automated deployment for static assets.
